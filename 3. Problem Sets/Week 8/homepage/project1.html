Educational Management System

| Languages/Frameworks: HTML/CSS, SQLite, PHP, JQuery, JavaScript, Bootstrap |
| Project Type: Group (Role - Project Leader), Class: Software Engineering I & II |
| FALL 2021 - SPRING 2022 |

  • Utilized all stages of the Software Development Life Cycle to create a university portal for documenting courses & student rosters, as well as taking attendance.
  • Implemented an account-based verification system (via codes and hashed passwords) for both instructor and student account types.
  • Student accounts have the ability to view their currently registered courses, as well as their fellow students within each given course. Whereas, Instructors can create courses, add or remove students from them, and generate attendance codes for each lesson in a course.

  • My work consisted of everything on the back-end, all prototyping in Figma, and all aspects of the following functionalities: Login; Sign-up; Course Creation; Adding/Removing/Searching Students; and Attendance Code Checking/Generation/Log.
